LadyCred

Quem queremos ajudar

Ajudar Mulheres que estão atuando na informalidade, principalmente que estão com o nome restrito, e com isso não podem ter opções de formas de pagamento para seus clientes.

Criar um APP com as seguintes soluções:

1 - Centralização de pagamento Online: Depósito Pagamento via Crédito e débito.

2 - Calculadora de gastos onde a usuário coloca: Matéria prima, Valor de venda e calcula tributos a serem pagos = Tem uma estimativa de valor de Lucro.

3 - Barra de Progresso (Gamification) onde é colocado um valor de Meta para o pagamento, inicialmente, de sua dívida para limpar o nome, e após, metas pessoais e até mesmo do seu negócio.

4 - Centralização de conteúdo online sobre educação financeira já existente, como SEBRAE, Blogs gerais de finanças.

5- Assistente virtual (Chatbot) para facilitar acesso a informações do conteúdo de Educação financeira.

Parcerias:

Com Instituições Financeiras para quando essas mulheres, quiserem sacar o valor depositado, gere uma ordem de pagamento para a retirada.

Além, nessa parceria captar possíveis fornecimentos de microcréditos.

Benefícios:

Dar opção de pagamento para os clientes dessa microempreendedora, gerar mais vendas;

Aumento de Score para que ela seja bem vista na "praça" (Microcrédito x Financiamentos x Cartão de Crédito Pessoal);

Acesso ao conteúdo online centralizado com base orientação financeira;

Visualização simplificada do Lucro: cálculo dos gastos com a matéria prima e Tributos.

Soluções tecnológicas:

Um app feita na interface em React Native e utilizando back end Django (Python);

Captação dos dados inseridos na plataforma mobile realizado via Google Analytics;

Utilização API de pagamentos (Recebimento dos clientes e de transações pessoais);

Utilização de API fornecido pelo parceiro Serasa, para consulta direta no APP do Score.

Para pagamentos, utilizar recurso de leitor de código de barras (Usando o saldo do APP);

Chatbot será realizado com a tecnologia Watson (IBM);

Usabilidade e Funcionamento:

O LadyCred é um aplicativo criado com intuito de ser simples e prático em sua utilização, fornecendo uma boa navegação e experiência ao usuário.

Foi utilizado para criação do app a avaliação Heurística de Nielsen, e foi baseada em alguns diretrizes que falaremos abaixo.

Correspondência entre sistema e mundo real: O app foi criado com estrutura textual, ícones e imagens acessíveis e de fácil entendimento ao usuário.

Liberdade e controle do usuário: Em todas as telas foi criado um botão de voltar para o início, a fim de cancelar uma ação que ele escolheu por engano, dando a ele liberdade em cancelar o processo a qualquer momento.

Consistência e padrões: A interface do app contém ícones padronizados para que o usuário não tenha que adivinhar se as palavras e ações significam coisas diferentes.

Reconhecer ao invés de lembrar: No app a carga de memorização de telas do app é desnecessária pois todos os ícones e opções são direcionais à sua necessidade.

Flexibilidade e Eficiência: A interface do app atende tanto usuários experientes, quanto os inexperientes, pois as informações estão dispostas de maneira simples.

Estética e Design: Evitamos o uso desnecessário e irrelevante de elementos visuais que possam confundir o usuário.

Funcionamento:

Login: É necessário que o usuário crie um usuário e senha para ter acesso as configurações da conta.

Tela Inicial: Na tela inicial tem as informações de saldo, e ícones que direcionam às transações e informações abaixo:

  • Recebimento: Gera boletos para serem enviados ao pagador ou exibe os dados para ser efetuado transferência para a conta.
  • Pagamento: Também gera boletos para que o usuário pague suas compras ou efetue a transferência para outra conta.
  • Saque: Emite uma ordem de pagamento para que o usuário vá até uma agência do banco e efetue o saque.
  • Metas: Nessa tela o usuário terá acesso a uma barra indicando quanto falta para ele cumprir a sua meta, que inicialmente será para quitação da negativação em seu nome.
  • Dicas: Nessa opção o usuário poderá acessar o menu onde são disponibilizados links de conteúdo já existentes sobre educação financeira.
  • Calculadora: Tela onde o usuário poderá inserir o valor gasto com os produtos x valor de venda e assim o lucro que terá já com a inclusão dos tributos.
  • Chatbot: Em todas as telas o usuário terá um botão de consultar as dicas de conteúdos de educação financeira.
